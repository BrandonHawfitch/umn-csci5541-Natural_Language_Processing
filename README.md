# Natural Language Processing (CSCI 5541)

### Graduate Academic Course | University of Minnesota
* **Language/Stack:** Python, Jupyter Notebooks
* **Core Focus:** Computational Linguistics, Sequence Modeling, and Deep Learning for Text

---

### Course Overview & Scope
This repository catalogs a series of comprehensive pipeline implementations covering foundational and state-of-the-art NLP techniques. The curriculum advanced progressively from standard text normalization and tokenization to building and tuning modern neural sequence networks.

### Key Work Completed
* **Text Processing Pipelines:** Implemented robust tokenization, stemming, n-gram language modeling, and vocabulary building strategies.
* **Sequence Tagging & Parsing:** Developed syntactic and semantic analysis scripts utilizing sequence labels (such as Part-of-Speech tagging).
* **Neural NLP Applications:** Built and trained deep-learning text classification and representation models across five major iterative frameworks (`hw1_nlp.ipynb` to `hw5_nlp.ipynb`).

### Skills & Technologies Applied
* **Machine Learning for Text:** Word Embeddings, Recurrent/Transformer Foundations, Sequence Classification.
* **Core NLP Concepts:** N-gram Language Models, Vector Space Models, Tokenization Dynamics, and Evaluation Metrics (Perplexity, F1-Score).
* **Tooling:** Python, PyTorch/TensorFlow, Scikit-Learn, Jupyter ecosystem.
